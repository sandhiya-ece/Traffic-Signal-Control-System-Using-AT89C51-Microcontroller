# Traffic Signal Control System Using AT89C51 Microcontroller

## Description
This project demonstrates a Traffic Signal Control System using the AT89C51 (8051) microcontroller with Embedded C. The system controls red, yellow, and green traffic signal LEDs while displaying the countdown timer on a single common cathode 7-segment display. The project is developed using Keil uVision and simulated in Proteus.

## Components Used
- AT89C51 Microcontroller
- Single Common Cathode 7-Segment Display
- Red LED
- Yellow LED
- Green LED

## Software Used
- Keil uVision
- Proteus 8
- Embedded C

## Files
- traffic_signal.c – Embedded C source code
- traffic_signal.uvproj – Keil project file
- STARTUP.A51 – Startup file
- traffic_signal.pdsprj – Proteus project file
- circuit.cng - proteus circuit diagram
